# Rivie's Moonlit Meowventure

Welcome El Padres Ninos to a game starring Rivie's choice of a cat, cheetah, turtle, penguin, lion, or tiger. Every level has a unique layout and enemy pattern. Collect pizza cupcakes and one invincibility cat-star per level, pass friendly heart cats for extra lives, and battle a boss every three levels.

The browser keeps a local top-10 leaderboard using each hero's entered name and final pizza-cupcake count. A friendly flamingo celebrates every completed level.

## Play

Open `index.html` in a modern web browser. No install or build step is needed.

- Move: Arrow keys or A / D
- Jump: Space, W, or Up Arrow
- Mobile: Large on-screen controls, buffered jumps, coyote-time forgiveness, and full-screen play

Collect pizza cupcakes, jump on the grumpy cats and T-rexes, reach each flag, and stomp the gorilla boss five times.
