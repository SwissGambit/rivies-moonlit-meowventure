# Rivie's Moonlit Meowventure

A cheerful ten-level browser platformer starring Rivie's choice of a cat, cheetah, turtle, penguin, lion, or tiger. Collect pizza cupcakes and one invincibility cat-star per level, dodge cats and T-rexes, and battle a boss every three levels.

## Play

Open `index.html` in a modern web browser. No install or build step is needed.

- Move: Arrow keys or A / D
- Jump: Space, W, or Up Arrow
- Mobile: On-screen left, right, and jump buttons

Collect pizza cupcakes, jump on the grumpy cats and T-rexes, reach each flag, and stomp the gorilla boss five times.
